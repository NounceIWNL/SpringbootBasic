Выпускная работа (демо на github.com)

1. Техническое задание (суть проекта в виде текста)
2. Структура базы данных ERD
3. Команды для создания базы данных DDL
4. Разработка дизайна Wed-приложения (карта сайта, макеты страниц) - design
5. Разработка интерфейса пользователя (html, css, js) - верстка
6. Разработка серверной части web-приложения
7. Интеграция интерфейса пользователя и серверной части (thymeleaf or fetch via JS)
8. Добавление функции многопользовательской среды (реализация безопасности Web-приложения)


Features

1. Свежая идея
2. Интуитивно понятный и удобный дизайн
3. Объединение таблиц в базах данных (joins)
4. Возможности расширения/усовершенствования приложения
5. Использование различных библиотек, технологий и т.д.
6. Документирование (по rest api , в частности, Swagger; комментарии в коде - Javadoc)
7. Тестирование (модульное, интеграционное, selenium, нагрузочная способность приложения)
